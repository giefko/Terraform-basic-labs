# Lab 3 - Storage Account

## Goal

This lab introduces the basic structure of an Azure Storage Account using Terraform.

## Resources

- Resource Group
- Storage Account

## Concepts covered

- Azure Storage Accounts
- Storage account tier
- Replication type
- Terraform resource structure
- Basic tagging

## Notes

This lab demonstrates how a basic Azure Storage Account can be defined in Terraform.  
It is intended for learning and validation purposes and focuses on simple storage configuration concepts commonly used in Azure environments.
