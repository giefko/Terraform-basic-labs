# Lab 4 - Network Interface

## Goal

This lab introduces Azure Network Interfaces (NICs) and demonstrates how they connect compute resources to a subnet in a virtual network.

## Resources

- Resource Group
- Virtual Network
- Subnet
- Network Interface

## Concepts covered

- Azure Network Interfaces
- Subnet attachment
- Private IP allocation
- Basic compute-to-network connectivity
- Terraform resource dependencies

## Notes

This lab demonstrates how a network interface can be attached to a subnet within a virtual network. It helps illustrate how Azure virtual machines connect to networking resources and is intended for learning and validation purposes.
