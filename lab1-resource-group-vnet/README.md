# Lab 1 - Resource Group and Virtual Network

## Goal

This lab introduces the basic structure of Azure networking resources using Terraform.

## Resources

- Resource Group
- Virtual Network
- Web Subnet
- App Subnet
- DB Subnet

## Concepts covered

- Azure Resource Groups
- Virtual Networks (VNets)
- Subnet segmentation
- Address spaces
- Basic infrastructure structure

## Notes

This lab is designed for learning and validation purposes. It helps demonstrate how Azure networking resources are structured in Terraform configuration files.
