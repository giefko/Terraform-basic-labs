# Lab 2 - Network Security Group

## Goal

This lab introduces Azure Network Security Groups (NSGs) and demonstrates how inbound traffic can be controlled using Terraform.

## Resources

- Resource Group
- Virtual Network
- Web Subnet
- Network Security Group
- HTTP rule
- HTTPS rule
- NSG association with subnet

## Concepts covered

- Network Security Groups
- Inbound security rules
- Rule priorities
- Subnet-level NSG association
- Basic Azure network security

## Notes

This lab builds on the networking concepts introduced in Lab 1 and shows how Azure traffic filtering rules can be applied to subnets.
